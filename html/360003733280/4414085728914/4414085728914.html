v22.2 - January 18, 2022

New Features and Improvements

  • This release includes general optimization improvements.

Fixed Bugs

  • Сursor cannot be placed after a bracket at the beginning of a line with a spellcheck underline. This has been fixed.

  • IME suggestions in Japanese overlapped the text being typed in a segment. This has been fixed.

  • Track changes are not being automatically displayed. This has been fixed.

  • ku_arab_iq is being displayed in target as LTR instead of RTL. This has been fixed.

v21.63 - January 4, 2022

New Features and Improvements

  • QA issues are highlighted within segments and can be mitigated with the inline menu or context menu.

Fixed Bugs

  • There was Inconsistent decimal representation between Memsource Editors and scorecards. This has been fixed.

  • The drop-down menu in Edit > Insert symbol > More symbols… was empty. This was been fixed.

  • Lock/unlock fails when a single segment is selected by its number. This has been fixed.

  • Font size preferences for panels were not working correctly when accessing a new window. This has been fixed.

  • Segment filter function is displaying incorrect results. This has been fixed.
