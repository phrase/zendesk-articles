Some file formats contain contextual information that aids in understanding and translation of those files. Memsource Editors display this information in the Context Note pane.

Plural form

In file formats supporting the plural form, some segments can be designated as plural variants. When detected, the Context pane automatically opens and information about the plural segment will be displayed:

  • Additional text if present in the original file

    Example:

    msgctxt in a PO Gettext file may be added to the Note field.

  • Plural Category (depending on the target language)

  • Plural Examples

There are three types of context information:

sidebar. Context Key

  • Automatically retrieved from the source file on import or defined manually for supported file types.

  • Supported in:

    • Chrome

    • CSV

    • Desktop Entry

    • DTD

    • INI

    • Joomla

    • JSON (limits)

    • Mac strings

    • MIF

    • Mozilla properties

    • Multilingual MS Excel

    • PO

    • properties

    • RESJSON

    • SRT

    • TS

    • Windows RC

    • XLIFF

    • XML (limits)

    • YAML (limits)

sidebar. Context Note

  • The Context Note allows users to specify sections of source documents where additional information relevant for translation is stored.

  • Set by the user in by clicking the File Import Settings in the Project Settings section.

  • Supported in:

    • Chrome

    • CSV

    • JSON (limits)

    • Mac Strings

    • PO

    • RESJSON

    • RESX

    • SBV

    • SRT

    • TS

    • Windows RC

    • SLIFF

    • XML (limits)

    • YAML (limits)

Filtering by Context Key and Context Note

Segments can be filtered by Context Key and Context Note. For more information, see Text Filter

sidebar. Context Origin

  • Retrieved from the original file on import. It includes information on the origin of the current segment. Example:

    footer will appear for a segment originating from the footer of a DOCX file.

  • Supported in:

    • Desktop Entry

    • MIF

    • TS

    • MS Office
