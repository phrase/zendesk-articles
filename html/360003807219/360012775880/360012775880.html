The Memsource Editors are where linguists perform translation activities.

Memsource has two editors with identical CAT functionalities:

  • Memsource Editor for Desktop

  • Memsource Editor for Web

These editors feature standard functions such as text filtering, find and replace, spellchecking (for supported languages), and autocomplete.

The Memsource Editor for Mobile is also available with more limited functionality.

The editors are divided into working areas, or panes:

  • CAT Pane

  • Search Pane

  • Quality Assurance Pane

  • Change Pane

  • In-Context Preview

  • More Pane

  • Context Note

If reference files are associated with a project, they will be indicated by a paper clip in the resource status indicators.

Appearance Preferences

To set Appearance Preferences, follow these steps:

  1. From the Tools menu, select Preferences.

    The PREFERENCES window opens.

  2. Select the Appearance tab.

    Available options:

    • Theme

    • Translation Table

    • Hidden Characters

    • Comments

    • Panels

  3. Select required options and click Save.

    Appearance preferences are saved for the editor.

Misc Preferences

To set Misc preferences, follow these steps:

  1. From the Tools menu, select Preferences.

    The PREFERENCES window opens.

  2. Select the Misc tab.

    Available options:

    • Auto-complete

    • Instant QA

    • Multiple instances of the Desktop Editor (Desktop Editor only)

  3. Click Save.

    Misc preferences are saved for the editor.

Status Bar

The status bar at the bottom of the Memsource Editor provides metadata related to word counts, user modifications, and the source file name. User information displayed is based on project settings.

  • Segments

    • Total number of segments/confirmed segments.

  • Words

    • Total source words/confirmed source words (For CJK languages with no space between words, the number of words will equal the number of characters excluding punctuation marks).

  • Chars

    • The first set of numbers is the total source characters/confirmed source characters. The second set is the character count for the source/target in the selected segment.

A detailed word count including 100% matches, fuzzy matches, etc., is provided by an analysis from within a project. For more information, see Analysis.

Editor Resource Status

The resource status indicators provide information about if and how the editor is connected to translation resources.

The indicators are presented in the bottom right of the desktop editor and the top right of the web editor.

Common error messages

No connection (Desktop Editor)

  • Ensure the server address (cloud.memsource.com, or the one provided by the Project Manager) is correct.

  • Check the proxy settings.

  • Check your Internet connection.

  • Use Winmtr (free download for Windows) to check the route from your computer to our server. Ensure your ISP is not re-routing your connection via too many servers and slowing it down.

Invalid username and/or password (Desktop Editor)

  • Check your username and password.

    Try to log in to your Linguist Portal with the same credentials.

    If password is forgotten, create a new one.

Unknown Error (Desktop Editor)

  • Lost connection to the server. Check internet connection.

Login is Green but All Other Indicators are Red (Desktop Editor)

  • User account was deactivated by the PM.

Server not responding (Desktop Editor)

When trying to Run QA checks.

  • Check the login indicators.

    Alternatively, upload a file to the server (click on Document and selecting Upload to server), and check the error message in the bottom right-hand corner.

Other errors (Desktop Editor)

Memsource Error Log

  • Memsource Editor for Desktop saves a log file on your local disk.

    Look for a folder .memsource, subfolder logs.

    In Windows, this can be located in C:\Users\your Windows username\.memsource\logs

    On Mac, it is in Applications/Memsource Editor/Show contents

    If contacting Memsource support, attach this log to the request.

You are not authorized to access this job

  • The job was removed from the server by the Project Manager (PM)—contact the PM.

  • The job was assigned to another Linguist—contact the PM.

  • You are trying to access a job using an incorrect username (if you have more then one).

TM, TB, or MT are Gray

  • They are not attached to the project in question. This could be intentional—contact the PM if unsure.

Error at 95% (or 97%)

When Uploading the Translation: Error at 95% (or 97%), Unable to upload translation— Access denied.

  • Job not accepted - Go to Linguist Portal and change the status.

  • The job is set to Completed—access no longer allowed. Contact the PM.

  • The job was assigned to another Linguist. Contact the PM.

Customize Shortcuts

Custom shortcuts can be added to the keyboard shortcuts in the Desktop Editor. All the default shortcuts for commands in the File, Edit, Tools, and Format menus can be customized.

To customize a shortcut, follow these steps:

  1. From the Tools menus, select Customize Shortcuts.

    The Customize Keyboard Shortcuts window opens.

  2. Select or Search for a shortcut to be customized.

  3. Click on the shortcut button.

    The shortcut changes to Press Keys.

  4. Type the new shortcut combination.

  5. Click Save.

    The custom shortcut is applied.
