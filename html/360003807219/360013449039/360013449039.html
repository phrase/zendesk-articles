Customize Shortcuts (Editor for Desktop)

Custom shortcuts can be added to the keyboard shortcuts in the Editor for Desktop. All the default shortcuts for commands in the File, Edit, Tools, and Format menus can be customized.

Some key combinations are reserved for system use and cannot be used to create custom shortcuts:

  • Shift+[key]

  • Alt+[character]

  • Ctrl+Tab, Ctrl+C, Ctrl+X, Ctrl+P

To customize a shortcut, follow these steps:

  1. From the Tools menus, select Customize Shortcuts.

    The Customize Keyboard Shortcuts window opens.

  2. Select or Search for a shortcut to be customized.

  3. Click on the shortcut button.

    The shortcut changes to Press Keys.

  4. Type the new shortcut combination.

  5. Click Save.

    The custom shortcut is applied.
