Templates and scripts used for the HC are stored in the Memsource GitHub instance. Access to the instance can be provided by Michal Kebrt if required.

Files that may require modification are:

  • script.js

    This file contains redirects. When the HC was rewritten, it was not possible to assign content published from Paligo to existing pages so many redirects needed to be created. This functionality now exists so more redirects should not be necessary unless deleting old and irrelevant content.

    To create a redirect, place the article id from the end of a retired page URL in var oldIds and the article id from the new page in var newIds.

    Warning

    Be very careful that the numbers line up correctly in the tables.

  • article_page.hbs

    As the file name suggests, this is the template for the main article page. Page functionality (Articles related, sidebar, helpfulness) and layout is defined here.

  • manifest.json

    Version control. To be updated with every change to the previously mentioned files.
