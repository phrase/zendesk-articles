The right hand drop down of Articles related to....:

The menu is defined in article_page.hbs maintained in the templates folder of the zendesk-theme directory on GitHub. The specific class is div class="related-to-label". If access is required, contact Michal Kebrt.

The content is defined with dynamic content items; each category having its own item.

Articles are assigned categories using an applied taxonomy in Paligo. When published, taxonomies are applied as labels on the HC articles.

To create a new category, follow these steps:

  1. Add the class to the article_page.hbs file copying the format of the existing classes.

  2. Create the dynamic content item using the same formatting as the existing items.

  3. Create the taxonomy in Paligo and apply it to the required articles.

  4. Publish the articles.
