v21.25 - January 4, 2022

New Features and Improvements

  • Marketo: Landing Pages, Emails and Email Templates can now be previewed in the editors.

  • GitLab: Translations can now be exported as pull requests.

  • API: TransMemories import now returns segment count.

Fixed Bugs

  • HubSpot:

    • After being exported back, the page title remained untranslated. This has been fixed.

    • A changed layout of a source page is not being transferred to an existing translation. This has been fixed.

    • Some content is triggering a false warning when exported. This has been fixed.

  • Google Drive: Files imported from a Google Shared drive were being exported back to a Personal drive. This has been fixed.
