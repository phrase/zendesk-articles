v21.22 - November 16, 2021

New Features and Improvements

  • Sitecore: Content can now be previewed in the Memsource Editor.

  • Zendesk: Connector did not send error when user did not have necessary access rights. The connector now displays an informative error message.

  • HubSpot: Added support for emails

Fixed Bugs

  • Zendesk:

    • Categories and sections display & as amp in the target. This has been fixed.

    • When using APC, source language is not being passed as a parameter. This has been fixed.

  • OneDrive: User receives a OneDrive account must be authorized message when trying to save the connector and files are not listed when trying to create jobs. This has been fixed.

  • HubSpot:

    • Formatting information is being imported to jobs as translatable content. This has been fixed.

    • Translation of already Published page is also Published when Save translation as is set to Draft. This has been fixed.

    • When A/B testing was used, only the master copy was pulled to Memsource. This has been fixed.

    • Some layout and page styling was missing in the translated pages. This has been fixed.

  • Contentful: Creating jobs from Contentful entry-level is generating wrong jobs names. This has been fixed.

  • Dropbox: Some file imports from the repo are failing. This has been fixed.

v21.21 - November 2, 2021

New Features and Improvements

  • Git Connector: Authentication options are now limited to either Password or Private key.

  • Contentful Connector: All reference entries are translated when the Localized reference field is multivalued.

Fixed Bugs

  • FTP Connector: Japanese characters in the filename were encoded incorrectly when imported. This has been fixed.

  • HubSpot:

    • Some content is imported twice along with layout/formatting information. This has been fixed.

    • Other form elements not imported for translation. This has been fixed.

    • CSS styles and names are injected as translatable text. This has been fixed.

  • GitLab: Untypical case of / used in branch name wasn’t handled and caused incorrect API call to GitLab. This has been fixed.

v21.20 - October 19, 2021

Fixed Bugs

  • Salesforce: The token is revoked if more than 5 connectors are created with the same account. This has been fixed.

  • GitHub: regexp for filename pattern doesn't work in APC when Include subfolders is enabled. This has been fixed.

  • API:

    • dateModified property in Get project template call response is in Unix epoch time format. This has been fixed.

    • There was a mismatch in NT matches between what is written in the API documentation and API response. This has been fixed.

v21.16 - August 24, 2021

v21.12 - June 29, 2021

New Features and Improvements

  • Microsoft OneDrive: Connector now supports OneDrive for Business.
