Human Translation Engines are hybrid AI/Human translation services. Memsource can use these platforms in the same manner as Machine Translation Engines.

Supported Engines:

  • Gengo

  • Unbabel

Set up Human Translation Engine

To set up a Human Translation Engine, follow these steps:

  1. From the SetupÂ  page, scroll down to the Integrations section and click on Human Translation Engines.

    The Human Translation Engines page opens.

  2. Click Create.

    The Create Human Translation Engine window opens.

  3. Select an engine from the dropdown list and click Create.

  4. Fill in the required fields and select options.

  5. Click Save.

    The selected engine is added to the list.

Submit Job to a Human Translation Engine

Only segments with empty target are sent for human translation. Pre-translate 101% and 100% matches from the TM first and only send the rest of the segments for human translation.

If the job contains many repetitions, lock all repetitions excepting the first to exclude them from human translation. This saves on translation costs and increases translation consistency.

To submit a job, follow these steps:

  1. From the jobs table, select job(s) for human translation.

  2. Click Pre-translate and select Human Translate where Empty....

    The Human Translate where Empty window opens.

  3. Select Engine. Hover over the information icon to display information about the selected engine.

    Options for the selected engine are displayed.

  4. Click Translate.

    The job is submitted and a spinning icon indicates the translation is in process. It can take 30 minutes or longer for the translated file to be received.

See Gengo or Unbabel documentation for more help.
